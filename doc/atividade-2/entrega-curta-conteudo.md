# Atividade 2 — Sistema de Entrada e Saída de Veículos

**Integrante:** Rafael Dreyer
**Disciplina:** Práticas Extensionistas III
**Curso:** Análise e Desenvolvimento de Sistemas — UNOESC

O projeto apresenta um sistema para cadastrar veículos e controlar entradas e saídas. Ele foi desenvolvido em PHP 8 com MySQL/MariaDB e possui login, página principal, cadastro de veículos, consulta e formulário de contato.
