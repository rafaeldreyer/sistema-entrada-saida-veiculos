## Descrição do projeto

O projeto tem como objetivo modelar um sistema de informação para controle de entrada e saída de veículos, permitindo o cadastro de usuários, condutores, veículos, vagas, movimentações e contatos. A modelagem servirá como base para o desenvolvimento futuro de um sistema em PHP 8 com MySQL.

## Estrutura do repositório

- `/documentacao`: contém o PDF final do trabalho e a versão editável em DOCX.

- `/diagramas/uml`: contém os diagramas UML do sistema.
